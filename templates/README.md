Reusable Markdown templates for every content type in this repository (fact, decision, person, project, event, vehicle, maintenance-record, purchase, recurring-bill, expense, source-record). To use one: copy the relevant template into the target directory, fill in the placeholders, remove the template comment, and set an accurate `classification` in the front matter. Default classification for this directory is **public**, since templates contain no real data.
